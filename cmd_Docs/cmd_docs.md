# Windows

## Move Files 
```bash
mv source destination
```

## Open System Properties
```bash
sysdm.cpl
```

## Open File Explorer
```bash
start .
```

## Open Control Panel
```bash
control
```

## Open Task Manager
```bask
taskmgr
```

## Open Chrome/Firefox
```bash
start chrome
start firefox
```

## Get All Users in the PC
```bash
net user
```

## Change user Password
```bash
net user USER_NAME *
```
